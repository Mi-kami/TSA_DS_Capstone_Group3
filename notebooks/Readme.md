📓 Notebooks Directory

This folder contains all Jupyter/Google Colab notebooks used throughout the project lifecycle.

Each notebook is structured according to a specific stage of the data science workflow, including:

- Project planning and overview

- Data collection

- Data cleaning and preprocessing

- Exploratory Data Analysis (EDA)

- Feature engineering

- Model development

- Model optimization

- Final evaluation

Notebooks are sequentially numbered to maintain logical progression and ensure clarity, reproducibility, and collaboration across the team.

This structure reflects industry best practices in organizing analytical workflows.
